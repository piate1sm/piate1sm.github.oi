Overview
Each of these project was done for a design class. The first was done in design bureau where we had to make designs for the business students national venture competition. The second was for a logo and identity class where we had to make a brand guide and products for a fictional non profit organization. The third was for an intro to print making class. This design was a linocut relief. The fourth is a box lid made in intro to ceramics. We had to make a box with a lid, bowls and cups. The last project is a website for into to web design were we had to make an informational website about a problem in the world.
